# SuperStore Sales Dashboard (Power BI)

An interactive Power BI dashboard project built to analyze sales performance, profit trends, and forecast future sales using time-series analysis.

This project demonstrates data visualization, business insights extraction, and 30-day sales forecasting, helping in better decision-making and trend prediction.

---

## Key Features

- 📈 Sales Overview – Total Sales, Orders, Profit, and Shipping Days
- 📊 Category-wise Analysis – Sales distribution by category and sub-category
- 💳 Payment Mode Insights – COD, Cards, and Online trends
- 🌍 Geographical Insights – Profit & Sales by State (Map visualization)
- 🚚 Shipping Analysis – Sales based on shipping modes
- 📅 Monthly Trends – Sales and Profit over time
- 🔮 30-Day Forecasting – Predictive analysis for future sales

---

## Forecasting Overview🔮

A dedicated forecasting module is implemented to predict future sales for the next 30 days based on historical data.

- Uses built-in Power BI forecasting techniques
- Helps identify future trends and demand patterns
- Useful for business planning and inventory management
